##### USB Drives
- **jetbot-043_nano-4gb-jp45.img**
	- OS image I believe we want. It is what the guide that Tingting gave us suggests.
- **JP461_sd-blob-b01.img**
	- OS image to use for the one time configuration that might be necessary as explained in the guide below.
- **debian-live-13.4.0-amd64-kde.iso**
	- Image of Debian that allows live booting into a KDE desktop environment. Use this if the other two failed to try and troubleshoot
- **archlinux-2026.03.01-x86_64.iso**
	- Image of Arch Linux in a headless setup, which means it only has a terminal and no desktop environment. If all else fails try this I guess? If even this is necessary we are doomed...
