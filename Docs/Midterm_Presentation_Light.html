Hardware & Software Stack

Hardware

  • JetAnk chassis (4WD tracked/wheeled)
  • Jetson Orin Nano Super — 8 GB RAM, 6-core ARM
  • Intel 8265 WiFi (wlP1p1s0)
  • Qwiic I2C bus (/dev/i2c-7)
  • CSI camera (to be mounted)

Software

  • JetPack 6.x / L4T R36.4.4 (Ubuntu 22.04)
  • Docker 28.2 — ML + display containers
  • TensorRT 10.3  ·  OpenCV 4.8
  • PyTorch / JupyterLab (via Docker)
  • pip installed via PyPA bootstrap (JetPack strips it from Python)

Task B — Camera Fusion for Perception

Fuse lane detection and object detection into a single per-frame state:

{
  "lane_offset": 0.10,   // deviation from center
  "lane_conf":   0.84,   // detection confidence
  "obstacle": { "class": "person", "conf": 0.92, "distance_m": 1.7 },
  "risk_level":   "high",
  "action_hint":  "stop"
}

Sources

  • Lane: notebooks/road_following/
  • Objects: YOLO via notebooks/object_following/

Evaluation

  • FPS under various resolutions
  • Accuracy under lighting / blur / occlusion
  • TensorRT optimisation if FPS too low

Task A — Camera Fusion for Driving (stretch goal)

Convert fused perception state into motor commands.

Perception StateAction
Lane centred, no obstacleMove forward
Lane offset detectedApply steering correction proportional to offset
High risk or low confidenceSlow down / stop
Obstacle within threshold distanceEmergency stop

Safety gates: confidence threshold · obstacle-distance threshold · manual e-stop

Deliverable: autonomous driving demo video
